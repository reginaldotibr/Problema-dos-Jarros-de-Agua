Problema dos Jarros de Água
Este projeto apresenta uma visualização didática do Problema dos Jarros de Água, um problema clássico da área de algoritmos, lógica e ciência da computação.
A solução é implementada utilizando HTML, CSS e JavaScript puro, com foco em clareza, organização e apoio ao ensino, permitindo acompanhar passo a passo as transferências de água entre os recipientes.

📘 Descrição do Problema
O problema consiste em três recipientes com capacidades diferentes:

Recipiente 1: 8 litros
Recipiente 2: 5 litros
Recipiente 3: 3 litros

Os recipientes não possuem marcações de medição, apenas a capacidade máxima conhecida.
Estado inicial

O recipiente de 8 litros inicia cheio
Os recipientes de 5 e 3 litros iniciam vazios

Objetivo

Obter exatamente 4 litros no recipiente de 8 litros
Obter exatamente 4 litros no recipiente de 5 litros

Regras

A água só pode ser transferida de um recipiente para outro
Cada transferência ocorre até que:

o recipiente de destino fique cheio, ou
o recipiente de origem fique vazio




🧠 Abordagem adotada
A solução utiliza um algoritmo de busca em espaço de estados, explorando as transferências possíveis entre os recipientes até que o estado desejado seja alcançado.
Cada passo da solução é exibido em uma tabela, mostrando:

A ação realizada
O volume atual de água em cada recipiente
A quantidade transferida entre os recipientes

Essa abordagem facilita a compreensão do funcionamento do algoritmo, especialmente em contextos educacionais.

🛠️ Tecnologias utilizadas

CSS3
JavaScript (puro, sem frameworks)
